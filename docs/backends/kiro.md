# Kiro

Kiro is Amazon's AI development environment. You can use it as a Tyde backend to get the Kiro experience inside Tyde's desktop workspace.

## Installation

Install the Kiro CLI if you haven't already:

```bash
curl -fsSL https://cli.kiro.dev/install | bash
```

Alternatively, open **Settings → Backend** in Tyde and click **Install** next to Kiro — Tyde will run the install script for you.

Once installed, make sure you've authenticated by running `kiro-cli` in your terminal at least once.

## Configuration

Select Kiro as your default backend in **Settings → Backend**. You can also select a model in the Kiro section of the settings panel.

## Usage tracking

Kiro tracks usage based on your plan. You can check your current usage in **Settings → Backend** under the Kiro section by clicking **Refresh usage**.
