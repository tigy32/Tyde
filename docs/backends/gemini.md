# Gemini CLI

Gemini CLI is Google's command-line coding agent. You can use it as a Tyde backend to get the Gemini experience inside Tyde's desktop workspace.

## Installation

Install the Gemini CLI if you haven't already:

```bash
npm install -g @google/gemini-cli
```

Alternatively, open **Settings → Backend** in Tyde and click **Install** next to Gemini — Tyde will run the npm install for you.

Once installed, make sure you've authenticated by running `gemini` in your terminal at least once.

## Configuration

Select Gemini as your default backend in **Settings → Backend**. You can select a model in the Gemini section of the settings panel. Gemini supports several model families including Gemini 2.5, Gemini 3, and Gemini 3.1.
