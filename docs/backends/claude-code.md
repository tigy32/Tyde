# Claude Code

Claude Code is Anthropic's CLI coding agent. You can use it as a Tyde backend to get the Claude Code experience inside Tyde's desktop workspace.

## Installation

Install the Claude Code CLI if you haven't already:

```bash
curl -fsSL https://claude.ai/install.sh | bash
```

Alternatively, open **Settings → Backend** in Tyde and click **Install** next to Claude Code — Tyde will run the install script for you.

Once installed, make sure you've authenticated by running `claude` in your terminal at least once.

## Configuration

Select Claude Code as your default backend in **Settings → Backend**. Claude Code uses your existing CLI configuration for model selection, permissions, and API key — there is nothing additional to configure in Tyde.
